# Spinno

[npm_url]: https://www.npmjs.org/package/spinno

Minimal React Spinner

## Installation

```bash
npm install --save spinno
```

## Usage

```jsx
import { Spinno } from 'spinno';

<Spinno color="#dd0000" size="20" />
```

## Defaults

- **size**: `16`
- **color**: `#333333`
- **animationDuration**: `2`
- **animationTimingFunction**: `linear`
